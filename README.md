# Women-Safety-App

Women Safety App is a mobile application developed using MIT App Inventor to demonstrate how mobile technology can be used to support personal safety and emergency communication. The application is designed to provide quick access to emergency assistance features through a simple and user-friendly interface.

The app allows users to register and manage emergency contacts who can be notified during critical situations. In an emergency, users can trigger an SOS alert that automatically sends their current GPS location to selected contacts through SMS. This enables family members, friends, or guardians to quickly identify the user's location and respond appropriately.

One of the key features of the application is the Shake-to-SOS functionality. By shaking the device, users can activate emergency alerts without navigating through menus, making it easier to use in stressful situations. The application can also initiate a direct phone call to a designated emergency contact, ensuring immediate communication when assistance is required.

To improve usability, emergency contact information is stored locally using TinyDB, allowing users to save and access contact details efficiently. The application integrates location services, messaging, and communication features to create a basic emergency response system suitable for demonstration and learning purposes.

The project showcases the practical implementation of mobile app development concepts, including sensor integration, GPS location tracking, SMS services, local data storage, and event-driven programming. It serves as an example of how mobile applications can be designed to address real-world challenges related to personal safety and emergency preparedness.

**Disclaimer:** This application has been developed solely for educational, academic, and demonstration purposes. It is a prototype intended to showcase technical concepts and mobile application development skills. The application has not undergone professional testing, certification, or validation for real-world emergency use. Therefore, it should not be relied upon as a primary safety solution in actual emergency situations. Users should always depend on official emergency services, verified safety applications, and local authorities for real-life safety and emergency assistance.
