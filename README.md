# 🛡️ Women Safety App

## 📱 Overview

Women Safety App is a mobile application developed using MIT App Inventor to demonstrate how mobile technology can support personal safety and emergency communication. The application is designed to provide quick access to emergency assistance features through a simple and user-friendly interface.

## ✨ Features

🔹 **Emergency Contact Management** – Register and manage trusted emergency contacts.

🔹 **SOS Alert System** – Send emergency alerts along with the user's current GPS location via SMS.

🔹 **📍 Live Location Sharing** – Helps emergency contacts identify the user's location during critical situations.

🔹 **📳 Shake-to-SOS Activation** – Trigger emergency alerts by shaking the device, enabling faster access in stressful situations.

🔹 **📞 Emergency Calling** – Instantly place a call to a designated emergency contact.

🔹 **💾 Local Storage (TinyDB)** – Securely store and retrieve emergency contact information on the device.

## 🛠️ Technologies Used

* MIT App Inventor
* GPS Location Services
* SMS Integration
* TinyDB Local Storage
* Accelerometer Sensor
* Event-Driven Programming

## 🎯 Project Objective

This project demonstrates the practical implementation of mobile app development concepts, including sensor integration, location tracking, messaging services, local data storage, and emergency communication features. It showcases how technology can be leveraged to address real-world safety concerns through accessible mobile solutions.

## ⚠️ Disclaimer

This application has been developed solely for **educational, academic, and demonstration purposes**. It is a prototype created to showcase technical concepts and mobile application development skills.

❗ The application has **not** undergone professional testing, certification, security auditing, or validation for real-world emergency use.

❗ It should **not** be relied upon as a primary safety solution during actual emergencies.
